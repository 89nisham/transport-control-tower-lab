# v1.0.0 Factory Release

Transport Control Tower Lab v1.0.0 is the local-first factory closeout release for Products 1-13.

## Release Scope

This release freezes the completed micro-product set:

- Product 1: Trip Sheet Doctor
- Product 2: GeoReplay
- Product 3: ETA Watch
- Product 4: DetentionClock
- Product 5: GateTruth
- Product 6: FuelGuard
- Product 7: UpdatePulse
- Product 8: DelayLens
- Product 9: PODPulse
- Product 10: LaneLab

## Local-First Boundary

The release stays file-based and demo-data friendly. It does not add paid APIs, authentication, databases, email, WhatsApp, Telegram, BI-server deployment, live telematics, TMS integrations, or SaaS workflow features.

## Validation Checklist

Run before tagging:

```bash
git fetch origin --tags
git status
git status --short
git branch --show-current
git rev-list --left-right --count origin/main...HEAD
gh run list --limit 5
uv sync
uv run pytest
uv run ruff check .
uv run python -m compileall -q .
uv run tower-brief tower_brief/demo_data tower_brief/output
```

Expected closeout state:

- Branch is `main`.
- Worktree is clean before release edits and clean after commit.
- Local `main` is aligned with `origin/main` before release edits.
- Latest GitHub Actions CI is green before release edits.
- `v0.13.0-tower-brief` points to `0db1771`.
- `v0.7.1-update-pulse` points to the spec-complete UpdatePulse commit.
- `v0.8.1-delay-lens` points to the spec-complete DelayLens commit.
- `v1.0.0` points to the factory closeout commit.
